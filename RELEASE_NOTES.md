# WinPilot Release Notes

---
## v1.3.8 (2026-07-05)

### 텍스트 추출 OCR 시작 안정화

- 앱 시작 전에 OCR 설치 작업을 기다리던 흐름을 제거해, OCR 언어팩 설치가 꼬이거나 오래 걸려도 프로그램 창이 뜨지 않는 문제를 방지
- OCR 언어팩 자동 구성 작업 전체를 예외 안전하게 감싸, 초기화 실패가 나도 앱 시작 자체는 막지 않도록 수정
- OCR 언어 지원 여부 판정도 예외 안전하게 보강해, 문제 PC에서 시작 단계가 죽지 않고 텍스트 추출 탭에서만 상태 문구로 원인을 확인할 수 있게 정리

---
## v1.3.7 (2026-07-05)

### 텍스트 추출 OCR 시작 단계 선설치

- 앱 실행 직후 OCR 언어팩 점검/설치를 바로 시작하고, **사용 가능한 OCR 엔진이 하나도 없는 경우**에는 메인 창 표시 전에 최대 5초 먼저 대기해 초기 설치를 앞당김
- 텍스트 추출 탭도 같은 OCR 사용 가능 판정 로직을 공유하도록 정리해, 시작 단계와 탭 진입 시 동작 기준이 어긋나지 않게 수정
- 기존처럼 설치 작업이 오래 걸려도 앱 전체가 장시간 멈추지 않도록 유지하면서, OCR 없는 PC에서는 설치가 더 빨리 붙도록 개선

---
## v1.3.6 (2026-07-05)

### 텍스트 추출 OCR 언어팩 자동 구성

- 앱 시작 시 **한국어/영어 OCR 언어팩** 상태를 확인하고, 부족한 항목은 자동으로 설치한 뒤 텍스트 추출이 바로 동작하도록 개선
- 텍스트 추출 탭에서 OCR 작업을 시작하면 언어팩 구성이 끝날 때까지 대기한 뒤 인식을 진행해, 실행 직후 붙여넣어도 실패 확률을 줄임
- 앱이 이번 실행에서 직접 설치한 OCR 언어팩만 추적해 **종료 시 자동 제거**하도록 처리
- 자동 설치 실패 시 원인을 상태 문구에 함께 표시해 어떤 언어팩 구성이 막혔는지 바로 확인 가능

## v1.3.5 (2026-07-05)

### 텍스트 추출 OCR 추가 보강

- OCR 전처리 전에 이미지를 **흰 배경으로 평탄화**해 투명 배경·다크 캡처에서 글자가 뭉개지는 문제를 줄임
- 색상 원본 / 명암 정규화 / 다중 임계값 흑백 / 반전 흑백까지 포함해 후보 이미지를 더 늘리고, 결과가 가장 좋은 OCR 결과를 자동 채택
- 작은 글씨 인식용 업스케일 범위를 한 번 더 높여 저해상도 캡처 대응을 강화

## v1.3.4 (2026-07-05)

### 텍스트 추출 OCR 인식 보강

- OCR 전처리를 단일 그레이스케일에서 **다중 전처리(그레이스케일 / 고대비 흑백 / 반전 흑백)** 방식으로 확장
- 작은 글씨·저대비 화면에서 인식률을 높이기 위해 업스케일 범위를 강화하고, 전처리 결과 중 **가장 인식 결과가 좋은 이미지**를 자동 선택
- 인식 실패 시 안내 문구를 보강해 더 큰 글씨·밝은 배경 재시도를 바로 유도

## v1.3.3 (2026-07-04)

### 업데이트 감지/실행 흐름 개선

- 앱을 켜 둔 상태에서도 백그라운드에서 최신 릴리스를 다시 확인하고, 새 버전이 나오면 **팝업 없이** 업데이트 버튼만 표시
- 사이드바 하단 업데이트 알림과 설정 탭의 버튼이 모두 **즉시 업데이트 실행 버튼**으로 동작하도록 변경
- 업데이트 감지 주기를 30분에서 **2분**으로 줄여, 배포 직후에도 실행 중인 앱에서 더 빨리 새 버전을 감지하도록 개선

## v1.3.2 (2026-07-02)

### Uties — 검진 SamFile 파서 탭 추가

- **검진 종류 / 검사 항목** 콤보박스에 따라 SamFile을 고정폭 필드 구조로 파싱해 항목:값 표로 정리
- 필드 타입 3종 지원(고정 길이 문자열 / 불리언 / 날짜) — 암검진 간암1·2차 스키마 내장
- SamFile 입력 / 정리 결과를 각각 독립된 카드형 그룹 패널로 분리, 스플리터로 폭 조절 가능

### 다크 테마 액센트 변경 (파랑 → 오렌지)

- 전체 앱 배경을 더 어둡게, 액센트 색상을 오렌지(#F97316) 계열로 통일

### 그리드 스타일 통일

- 이벤트 뷰어 / 작업 관리자 / Uties 정리 결과 등 모든 DataGrid에서 얼터네이팅(줄무늬) 행 배경 제거
- 얇은 구분선만으로 행을 나누도록 통일, 행 높이 38px로 통일

### 작업 관리자 — 프로세스 그룹핑 개선

- 프로세스 그룹을 이름 대신 앱 트리(루트 PID) 단위로 묶도록 변경
- 앱/백그라운드 프로세스 섹션 분리(그룹핑 → 헤더 행 방식으로 변경, 컬럼 뭉개짐 수정) + 선택 유지
- 그룹 확장 화살표 상시 표시, 선택색을 테마 반응형 오렌지로 통일
- 메모리 열을 작업 관리자와 동일한 활성 개인 작업 집합 기준으로 변경

## v1.3.0 (2026-06-23)

### 이미지 텍스트 추출 개편

- **OCR 인식률 향상**: 인식 전 이미지 전처리 추가 (작은 이미지 최대 3배 업스케일 + 그레이스케일 변환)
- **AI 교정 엔진을 Claude → DeepSeek로 변경**
  - OCR 완료 후 자동으로 교정 실행 (별도 버튼 불필요)
  - 1단계 규칙 교정(리가처·한글 깨짐 복원) → 2단계 DeepSeek 문맥 교정
  - API 키 미설정 시 규칙 교정까지만 적용 (현상 유지)
- **UI 정리**: 엔진 선택 콤보박스·교정 버튼·탭 내 API 키 입력란 제거 → 화면 간결화

### 설정 탭 — DeepSeek API 키 관리

- 설정 탭에서 DeepSeek API 키 저장 / 삭제
- **관리자 전용**: 저장·삭제 시 관리자 비밀번호 확인
- **마스킹 표시**: 키는 항상 ●●● 로 표시 (평문 노출 없음)
- **암호화 저장**: DPAPI(현재 Windows 계정 전용)로 암호화하여 저장
- 재시작 후에도 키 유지, 배포 시 기본 키 내장 지원

### 네비게이션 메뉴 명칭 변경

- 대시보드 → **홈**
- Ping 테스트 → **Ping Test**
- 파서 → **Converter**
- 이미지 텍스트 → **텍스트 추출**

## v1.2.0 (2026-06-22)

### 새 기능

- **이미지 텍스트 추출 탭 신규 추가**
  - 스크린샷 캡처 후 Ctrl+V 붙여넣기 → 텍스트 자동 인식 (Windows OCR)
  - 한국어 + 영어 이중 엔진 병렬 실행으로 혼합 텍스트 인식률 향상
  - 이미지 내 텍스트 위치(BoundingRect) 기반으로 공백·줄바꿈 재구성
  - **번역 기능**: 한국어·영어·일본어·중국어·베트남어 등 8개 언어 지원 (Google Translate)
  - **텍스트 교정 기능**
    - 규칙 기반: 리가처 복원(ﬁ→fi), 한글+CJK 합성(계조卜번호→계좌번호), 오인식 치환
    - Claude AI 교정: API 키 설정 시 문맥 기반 OCR 오류 자동 수정
  - **이전 이미지 선택 팝업**: 세션 내 붙여넣은 이미지 최대 5개 이력 관리
  - **이미지 확대 오버레이**: 현재 이미지 전체화면 크기로 확대 확인 가능
  - 복사 버튼 클릭 시 시각 피드백 ("복사되었습니다" 2.5초 표시 + 버튼 깜박)

### 개선

- **SFC /scannow 결과 표시 개선**
  - 실행 중: 초 단위 경과 시간 표시 (SFC는 완료 전까지 출력 없음)
  - 완료 후: 100줄 재생 대신 핵심 결과 한 줄 + 소요 시간만 표시
