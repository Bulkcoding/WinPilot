# WinPilot Release Notes

## v1.2.0 (2026-06-22)

### 새 기능

- **이미지 텍스트 추출 탭 신규 추가**
  - 스크린샷 캡처 후 Ctrl+V 붙여넣기 → 텍스트 자동 인식 (Windows OCR)
  - 한국어 + 영어 이중 엔진 병렬 실행으로 혼합 텍스트 인식률 향상
  - 이미지 내 텍스트 위치(BoundingRect) 기반으로 공백·줄바꿈 재구성
  - **번역 기능**: 한국어·영어·일본어·중국어·베트남어 등 8개 언어 지원 (Google Translate)
  - **텍스트 교정 기능**
    - 규칙 기반: 리가처 복원(ﬁ→fi), 한글+CJK 합성(계조卜번호→계좌번호), 오인식 치환
    - Claude AI 교정: API 키 설정 시 문맥 기반 OCR 오류 자동 수정
  - **이전 이미지 선택 팝업**: 세션 내 붙여넣은 이미지 최대 5개 이력 관리
  - **이미지 확대 오버레이**: 현재 이미지 전체화면 크기로 확대 확인 가능
  - 복사 버튼 클릭 시 시각 피드백 ("복사되었습니다" 2.5초 표시 + 버튼 깜박)

### 개선

- **SFC /scannow 결과 표시 개선**
  - 실행 중: 초 단위 경과 시간 표시 (SFC는 완료 전까지 출력 없음)
  - 완료 후: 100줄 재생 대신 핵심 결과 한 줄 + 소요 시간만 표시
